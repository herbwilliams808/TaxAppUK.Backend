#### Box A4: Total pay
```
A4 = A1 + A2 + A3

A4(TotalPay) =
A1(
    TaxablePayToDate +
    Tips
) +
A2(Not available) +
A3(
    taxableLumpSumsAndCertainIncome > amount +
    benefitFromEmployerFinancedRetirementScheme > amount
)
```
